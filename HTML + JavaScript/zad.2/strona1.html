Język Java

Java to obiektowy język programowania, który opiera się na klasach. Jest językiem ogólnego zastosowania Można pisać w nim zarówno aplikacje desktopowe,
jak i aplikacje webowe. Java wyróżnia się silnym typowaniem. Należą tu: obiektowość,dziedziczenie,obsługa programowania rozproszonego,
niezależność od architektury,bezpieczeństwo i niezawodność.
